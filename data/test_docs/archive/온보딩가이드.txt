## 3. 시스템 계정 설정

### 3.1 필수 계정

첫날 IT팀에서 다음 계정을 생성해드립니다:

| 시스템 | 용도 | URL |
|-------|------|-----|
| Google Workspace | 이메일, 캘린더, 드라이브 | mail.google.com |
| FlowSync | 업무 협업 | app.flowsync.io |
| Slack | 메신저 | techflow.slack.com |
| HR Portal | 인사 시스템 | hr.techflow.kr |
| GitHub | 소스 코드 (개발팀) | github.com |

### 3.2 계정 설정 순서

1. **이메일 확인**: welcome@techflow.kr에서 온 메일 확인
2. **비밀번호 설정**: 각 시스템 초기 비밀번호 변경
3. **2단계 인증**: 필수 시스템 2FA 설정
4. **프로필 완성**: 사진, 연락처 정보 입력

### 3.3 비밀번호 규칙

- 최소 12자 이상
- 대문자, 소문자, 숫자, 특수문자 포함
- 이전 비밀번호와 다르게
- 90일마다 변경

## 5. 업무 도구

### 5.1 FlowSync 사용법

TechFlow의 자체 협업 도구입니다. 모든 업무가 여기서 이루어집니다.

**주요 기능**:
- FlowDocs: 문서 작성 및 공동 편집
- FlowTask: 태스크 관리
- FlowDrive: 파일 저장 및 공유

**시작하기**:
1. app.flowsync.io 접속
2. 회사 계정으로 로그인
3. 팀 워크스페이스 참여

### 5.2 Slack 채널

| 채널 | 용도 |
|-----|------|
| #general | 전사 공지 |
| #random | 자유 대화 |
| #tech | 기술 논의 |
| #help-it | IT 지원 요청 |
| #help-hr | 인사 문의 |
| #lunch | 점심 모임 |

### 5.3 개발 환경 (개발팀)

**필수 설치**:
- Git
- VS Code (또는 선호 IDE)
- Docker
- Node.js / Python (프로젝트별)

**레포지토리 접근**:
1. GitHub 계정 생성 (개인 계정 가능)
2. 팀장에게 권한 요청
3. SSH 키 등록

## 8. 버디 프로그램

### 8.1 버디란?

신입사원의 적응을 돕는 기존 팀원입니다. 첫 3개월간 1:1로 멘토링합니다.

### 8.2 버디 역할

- 업무 관련 질문 응대
- 회사 문화 안내
- 점심/커피 함께하기
- 주 1회 1:1 체크인

### 8.3 버디 미팅

**권장 일정**:
- 첫 주: 매일 15분
- 2-4주: 주 3회
- 2-3개월: 주 1회

**Q: 비밀번호를 잊어버렸어요.**
A: IT 헬프데스크 (it@techflow.kr 또는 내선 1111)로 연락하세요.

**Q: 재택근무는 언제부터 가능한가요?**
A: 수습 종료 후 (3개월) 신청 가능합니다.

**Q: 점심은 어떻게 하나요?**
A: 법인카드로 결제하거나, 식대를 월말에 정산받습니다.

**Q: 야근 식대는 지원되나요?**
A: 네, 19시 이후 퇴근 시 15,000원 지원됩니다.
