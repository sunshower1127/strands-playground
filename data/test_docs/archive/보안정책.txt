> 버전: 2.0

## 1. 목적 및 적용 범위

### 1.1 목적

본 정책은 TechFlow Inc.의 정보자산을 보호하고, 정보보안 사고를 예방하며, 법적 규제를 준수하기 위해 수립되었습니다.

### 1.2 적용 범위

- 모든 임직원 (정규직, 계약직, 인턴)
- 협력업체 및 외주 인력
- 회사 시스템에 접근하는 모든 사용자

### 1.3 관련 법규

- 개인정보보호법
- 정보통신망법
- 전자금융거래법 (해당 시)

## 2. 정보 분류

### 2.1 분류 등급

| 등급 | 설명 | 예시 |
|-----|------|-----|
| 극비 (Confidential) | 유출 시 회사에 심각한 피해 | 고객 개인정보, 영업비밀, 소스코드 |


## 3. 접근 통제

### 3.1 계정 관리

**계정 생성**:
- 인사팀 입사 통보 → IT팀 계정 생성
- 최소 권한 원칙 적용

**계정 삭제**:
- 퇴사 당일 모든 계정 비활성화
- 7일 내 완전 삭제

**계정 검토**:
- 분기별 미사용 계정 점검
- 권한 적정성 검토

### 3.2 비밀번호 정책

| 항목 | 기준 |
|-----|------|
| 최소 길이 | 12자 이상 |
| 복잡도 | 대/소문자, 숫자, 특수문자 포함 |
| 변경 주기 | 90일 |
| 재사용 제한 | 최근 5개 사용 불가 |
| 잠금 | 5회 실패 시 30분 잠금 |

### 3.3 다중 인증 (MFA)

다음 시스템은 2단계 인증이 필수입니다:
- 이메일 (Google Workspace)
- 업무 시스템 (FlowSync)
- 코드 저장소 (GitHub)
- 클라우드 콘솔 (AWS, GCP)
- VPN

## 6. 데이터 보안

### 6.1 암호화

**전송 중 암호화**:
- 모든 통신 TLS 1.2 이상
- 이메일 첨부: 민감정보는 암호화 압축

**저장 시 암호화**:
- 데이터베이스: AES-256
- 파일 서버: 암호화 스토리지

### 6.2 백업

- 일일 백업: 주요 시스템
- 주간 백업: 전체 데이터
- 보관: 3개월 (규정에 따라 연장)

### 6.3 데이터 삭제

**일반 삭제**:
- 휴지통 이동 → 30일 후 자동 삭제

**보안 삭제**:
- 퇴사자 데이터: 90일 보관 후 완전 삭제
- 민감정보: 보안 삭제 도구 사용

## 7. 이메일 보안

### 7.1 피싱 주의

다음 이메일은 피싱일 수 있습니다:
- 긴급 조치 요구
- 비밀번호 재설정 요청
- 첨부파일 실행 요구
- 출처 불명의 링크

**의심 시 조치**:
1. 클릭하지 않기
2. 보안팀에 신고 (security@techflow.kr)
3. 피싱 신고 버튼 사용

### 7.2 외부 메일 주의

- 외부 발신 메일에 [EXTERNAL] 태그 표시
- 첨부파일 열기 전 발신자 확인
- 의심스러운 링크 클릭 금지

### 7.3 민감정보 발송

고객 개인정보, 재무정보 등 민감정보 발송 시:
1. 암호화 압축 (비밀번호 별도 전달)
2. 수신자 이메일 재확인
3. 가능한 보안 파일 공유 시스템 사용

## 8. 소프트웨어 보안

### 8.1 승인된 소프트웨어

회사에서 승인한 소프트웨어만 설치 가능합니다.

**기본 설치 소프트웨어**:
- Microsoft Office / Google Workspace
- Slack
- Zoom
- VS Code (개발팀)
- 브라우저 (Chrome, Safari)

**설치 요청**:
IT 헬프데스크에 요청서 제출

### 8.2 금지 소프트웨어

- P2P 파일 공유 프로그램
- 원격 제어 프로그램 (승인 없이)
- 크랙/불법 소프트웨어
- 악성코드 위험 프로그램

### 8.3 보안 업데이트

- OS 업데이트: 자동 업데이트 활성화
- 앱 업데이트: 알림 시 즉시 적용
- 긴급 패치: IT팀 공지 시 24시간 내 적용

## 9. 모바일/BYOD 보안

### 9.1 업무용 모바일

회사 지급 모바일 기기:
- MDM (모바일 관리) 설치 필수
- 화면 잠금 설정 필수 (PIN 6자리 이상)
- 분실 시 원격 초기화 동의

### 9.2 개인 기기 (BYOD)

개인 기기로 업무 시스템 접근 시:
- 2단계 인증 필수
- 화면 잠금 필수
- 탈옥/루팅 기기 금지
- 공용 Wi-Fi 접속 금지 (VPN 사용)
