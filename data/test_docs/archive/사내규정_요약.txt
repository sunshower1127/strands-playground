## 1. 근로 시간

### 1.1 기본 근무시간

- **정규 시간**: 09:00 ~ 18:00 (1일 8시간)
- **점심시간**: 12:00 ~ 13:00 (1시간)
- **주간 근로**: 40시간

### 1.2 탄력근무제

희망 시 다음 시간대로 변경 가능:
- 조기 출근: 08:00 ~ 17:00
- 지연 출근: 10:00 ~ 19:00

신청: HR Portal > 근태 > 탄력근무 신청

### 1.3 시간외 근무

- 야근 시 사전 승인 필요
- 야근 식대: 15,000원 (19시 이후)
- 주말/공휴일 근무: 대체휴무 또는 수당

## 2. 복장 규정

### 2.1 기본 원칙

**Smart Casual** 복장 권장

**허용**:
- 청바지, 면바지, 치노
- 셔츠, 블라우스, 니트
- 스니커즈, 로퍼, 플랫슈즈

**지양**:
- 운동복, 슬리퍼
- 과도한 노출
- 정장 필수 아님

### 2.2 예외 상황

- 외부 미팅/고객 방문: 비즈니스 캐주얼 권장
- 공식 행사: 별도 안내

---

## 3. 출장

### 3.1 국내 출장

**신청 절차**:
1. HR Portal > 출장 신청
2. 팀장 승인
3. 법인카드 사용

**지원 항목**:
| 항목 | 기준 |
|-----|------|
| 교통비 | 실비 (기차: KTX, 버스: 우등) |
| 숙박비 | 1박 15만원 한도 |
| 식비 | 1일 5만원 |
| 일비 | 2만원 |

### 3.2 해외 출장

**신청 절차**:
1. 출장 계획서 작성
2. 팀장 + 부서장 승인
3. 인사팀 항공권/숙소 예약

**지원 항목**:
| 항목 | 기준 |
|-----|------|
| 항공 | 이코노미 (10시간 이상: 비즈니스) |
| 숙박 | 도시별 기준 상이 |
| 식비 | 도시별 기준 상이 |
| 일비 | $50/일 |

## 4. 법인카드

### 4.1 사용 범위

**허용**:
- 업무 관련 식비
- 출장 경비
- 업무용 물품 구매
- 고객/파트너 접대

**불가**:
- 개인 용도
- 주류 (접대 제외)
- 유흥 업소

### 4.2 정산

- 월말까지 영수증 제출
- HR Portal에서 사용 내역 등록
- 미정산 시 급여 공제

## 6. 회의실 예약

### 6.1 예약 방법

FlowSync 캘린더에서 예약:
1. 새 일정 생성
2. 장소 > 회의실 선택
3. 저장

### 6.2 회의실 목록

|-----|-----|---------|
| 회의실 A | 1층 | 10명 |
| 회의실 B | 1층 | 6명 |
| 회의실 C | 2층 | 8명 |
| 대회의실 | 4층 | 30명 |
| 포커스룸 | 각 층 | 2명 |

- 정기 회의는 반복 예약 가능
