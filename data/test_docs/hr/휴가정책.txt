# TechFlow Inc. 휴가 정책

> 최종 수정: 2024년 1월 15일
> 담당: 인사팀 (hr@techflow.kr)

## 1. 연차 휴가

### 1.1 연차 휴가 일수

입사 연차에 따른 연차 휴가 일수는 다음과 같습니다:

| 근속 연수 | 연차 휴가 일수 |
|----------|--------------|
| 1년 미만 | 월 1일 (최대 11일) |
| 1년 이상 ~ 3년 미만 | 15일 |
| 3년 이상 ~ 5년 미만 | 17일 |
| 5년 이상 ~ 10년 미만 | 20일 |
| 10년 이상 | 25일 |

연차 휴가는 매년 1월 1일에 부여되며, 입사 첫 해에는 입사일 기준으로 월할 계산됩니다. 예를 들어 7월 1일 입사자는 해당 연도에 6일의 연차가 부여됩니다.

### 1.2 연차 사용 규정

- **사전 신청**: 연차 사용 3일 전까지 HR 시스템에서 신청
- **장기 휴가**: 5일 이상 연속 사용 시 2주 전 팀장 승인 필요
- **연말 소진**: 미사용 연차는 이월 불가, 12월 31일까지 사용 권장
- **연차 수당**: 퇴직 시 미사용 연차에 대해 수당 지급

### 1.3 연차 촉진제

회사는 연차 사용을 적극 권장합니다. 10월 1일 기준 잔여 연차가 5일 이상인 경우, 인사팀에서 사용 촉진 안내를 발송합니다. 11월 30일까지 사용 계획을 제출하지 않으면 회사가 지정한 날짜에 연차가 사용된 것으로 처리될 수 있습니다.

## 2. 병가

### 2.1 유급 병가

- 연간 10일의 유급 병가 제공
- 3일 이상 연속 사용 시 의사 진단서 제출 필요
- 병가는 연차와 별도로 관리됨

### 2.2 병가 신청 절차

1. 당일 오전 9시 이전 팀장에게 연락 (전화 또는 메신저)
2. HR 시스템에서 병가 신청
3. 3일 이상인 경우 복귀 시 진단서 제출

### 2.3 장기 병가

입원 등으로 10일 이상 병가가 필요한 경우:
- 인사팀과 상담 후 무급 병가 또는 휴직 전환 가능
- 4대 보험 관련 사항 별도 안내

## 3. 경조사 휴가

경조사 발생 시 다음과 같은 유급 휴가를 제공합니다:

| 구분 | 휴가 일수 | 경조금 |
|-----|---------|-------|
| 본인 결혼 | 5일 | 100만원 |
| 자녀 결혼 | 2일 | 50만원 |
| 부모 회갑 | 1일 | 30만원 |
| 배우자/본인 부모 사망 | 5일 | 100만원 |
| 조부모/배우자 부모 사망 | 3일 | 50만원 |
| 형제자매 사망 | 3일 | 30만원 |
| 자녀 출산 | 10일 (남성) | - |

경조사 휴가는 해당 사유 발생일 기준 전후로 사용해야 하며, 증빙 서류(청첩장, 사망진단서 등) 제출이 필요합니다.

## 4. 공가

### 4.1 공가 사유

다음의 경우 공가로 처리됩니다:
- 예비군/민방위 훈련
- 국민투표, 선거 참여
- 법원 출두 (증인, 감정 등)
- 건강검진 (회사 지정)

### 4.2 신청 방법

공가 사유 발생 전 관련 서류를 첨부하여 HR 시스템에서 신청합니다.

## 5. 특별 휴가

### 5.1 리프레시 휴가

장기 근속자에게 다음과 같은 리프레시 휴가를 부여합니다:

| 근속 연수 | 리프레시 휴가 |
|----------|-------------|
| 5년 | 3일 + 휴가비 50만원 |
| 10년 | 5일 + 휴가비 100만원 |
| 15년 | 7일 + 휴가비 150만원 |

리프레시 휴가는 해당 연도 내에 사용해야 하며 이월되지 않습니다.

### 5.2 포상 휴가

분기별 우수 사원에게는 포상 휴가 2일이 부여됩니다. 포상 휴가는 부여일로부터 6개월 이내에 사용해야 합니다.

## 6. 휴가 신청 시스템

### 6.1 HR Portal 사용법

1. hr.techflow.kr 접속
2. 사번 및 비밀번호로 로그인
3. [휴가 관리] > [휴가 신청] 메뉴 클릭
4. 휴가 종류, 기간, 사유 입력
5. 승인자 선택 후 제출

### 6.2 모바일 앱

TechFlow HR 앱 (iOS/Android)에서도 휴가 신청이 가능합니다. 앱스토어에서 "TechFlow HR"을 검색하여 설치하세요.

### 6.3 승인 절차

- 일반 연차: 팀장 승인
- 5일 이상 연속 휴가: 팀장 + 부서장 승인
- 병가 (3일 이상): 팀장 승인 + 진단서 첨부

## 7. 자주 묻는 질문 (FAQ)

**Q: 입사 첫 해에 연차를 모두 사용할 수 있나요?**
A: 네, 부여된 연차 내에서 자유롭게 사용 가능합니다. 단, 수습 기간(3개월) 중에는 팀장과 협의가 필요합니다.

**Q: 반차 사용이 가능한가요?**
A: 가능합니다. 오전 반차(09:00-13:00), 오후 반차(14:00-18:00)로 구분됩니다.

**Q: 휴가 중 긴급 업무가 발생하면 어떻게 하나요?**
A: 휴가 중 업무 복귀 요청 시 해당 일수는 연차에서 차감되지 않습니다. 대체 휴무로 보상됩니다.

**Q: 연차 수당은 어떻게 계산되나요?**
A: 미사용 연차 1일당 통상임금의 100%가 지급됩니다. 퇴직 시에만 적용됩니다.
