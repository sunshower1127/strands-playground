# TechFlow Inc. 재택근무 가이드

> 최종 수정: 2024년 3월 1일
> 담당: 인사팀 근태관리 (hr@techflow.kr)

## 1. 재택근무 제도 개요

### 1.1 도입 배경

TechFlow는 2020년 COVID-19 팬데믹 이후 재택근무 제도를 도입하였으며, 현재는 하이브리드 근무 형태로 정착되었습니다. 직원들의 워라밸 향상과 업무 효율 증진을 위해 유연한 근무 환경을 제공하고 있습니다.

### 1.2 적용 대상

재택근무는 다음 조건을 충족하는 직원에게 적용됩니다:

- 수습 기간(3개월) 종료 후 정규직
- 업무 특성상 재택근무가 가능한 직무
- 팀장 승인을 받은 직원

다음의 경우는 재택근무가 제한됩니다:
- 고객 대면 업무 담당자
- 보안 등급 A 업무 담당자
- 신입사원 (입사 후 3개월간)

### 1.3 재택근무 유형

| 유형 | 설명 | 신청 방법 |
|-----|-----|---------|
| 정기 재택 | 주 2일 고정 재택근무 | 월초 HR 시스템에서 요일 지정 |
| 수시 재택 | 필요시 단발성 재택근무 | 전일 18시까지 신청 |
| 풀 리모트 | 전일 재택근무 (특수 승인) | 부서장 + 인사팀장 승인 |

## 2. 재택근무 신청 절차

### 2.1 정기 재택근무 신청

매월 1일~5일 사이에 해당 월의 재택근무 요일을 지정합니다.

1. HR Portal 접속 (hr.techflow.kr)
2. [근태관리] > [재택근무 신청] 클릭
3. 재택근무 요일 선택 (최대 주 2일)
4. 신청 사유 입력
5. 팀장 승인 대기

**주의사항**:
- 팀 미팅이 있는 요일은 출근 권장
- 부서별 최소 출근율(60%) 유지 필요
- 월요일/금요일 재택은 팀당 2명 이내

### 2.2 수시 재택근무 신청

급하게 재택근무가 필요한 경우 전일 18시까지 신청합니다.

1. HR Portal 또는 모바일 앱 접속
2. [재택근무] > [수시 신청] 선택
3. 날짜 및 사유 입력
4. 팀장 승인 (당일 09시까지)

수시 재택은 월 4회로 제한됩니다.

### 2.3 풀 리모트 신청

특별한 사유로 장기간 풀 리모트가 필요한 경우:

- 신청 자격: 근속 2년 이상, 최근 인사평가 B등급 이상
- 승인 절차: 팀장 → 부서장 → 인사팀장
- 기간: 최대 6개월 (연장 가능)
- 사유: 육아, 간병, 이사 등

## 3. 재택근무 시 규정

### 3.1 근무 시간

재택근무 시에도 정규 근무시간(09:00~18:00)을 준수합니다.

- **출근 체크**: 09:00까지 Slack #attendance 채널에 "출근" 메시지
- **점심시간**: 12:00~13:00 (1시간)
- **퇴근 체크**: 업무 종료 시 Slack에 "퇴근" 메시지

탄력근무제 신청자는 합의된 시간에 출퇴근 체크합니다.

### 3.2 업무 보고

재택근무 시 다음과 같이 업무 현황을 공유합니다:

**일일 업무 보고**
- 시점: 퇴근 시
- 채널: Slack #daily-standup
- 형식:
  ```
  [오늘 한 일]
  - 작업 1
  - 작업 2

  [내일 할 일]
  - 작업 1

  [이슈/블로커]
  - (있으면 기재)
  ```

### 3.3 연락 가능 시간

재택근무 중에는 근무시간 내 즉시 연락 가능한 상태를 유지해야 합니다.

- Slack 상태: "재택근무 중" 으로 설정
- 응답 시간: 메시지 수신 후 15분 이내 확인
- 화상회의: 요청 시 10분 이내 참여 가능

점심시간이나 잠시 자리를 비울 때는 Slack 상태를 업데이트합니다.

### 3.4 회의 참석

재택근무 중에도 모든 예정된 회의에 참석해야 합니다.

- 화상회의 도구: Google Meet (기본), Zoom (외부 미팅)
- 카메라: 가급적 켜고 참여 권장
- 배경: 가상 배경 사용 가능 (회사 로고 배경 제공)

중요 회의 (부서 회의, 전사 미팅 등)는 가능한 출근하여 참석합니다.

## 4. 재택근무 환경 가이드

### 4.1 필수 장비

재택근무를 위해 다음 장비가 필요합니다:

| 항목 | 지급/지원 |
|-----|---------|
| 노트북 | 회사 지급 |
| 모니터 | 24인치 1대 지급 (희망 시) |
| 키보드/마우스 | 회사 지급 |
| 웹캠/헤드셋 | 회사 지급 |
| 인터넷 | 월 5만원 지원 |

장비 신청: IT 헬프데스크 (it@techflow.kr)

### 4.2 권장 환경

효율적인 재택근무를 위해 다음 환경을 권장합니다:

**인터넷**
- 최소 100Mbps 이상
- 유선 연결 권장 (화상회의 품질)

**작업 공간**
- 독립된 업무 공간 확보
- 적절한 조명 및 환기
- 의자/책상 높이 조절

**보안**
- VPN 연결 필수
- 공용 Wi-Fi 사용 금지
- 가정용 공유기 비밀번호 설정

### 4.3 재택근무 지원금

재택근무자에게 다음과 같은 지원금을 제공합니다:

- 인터넷 요금: 월 50,000원
- 전기요금: 월 30,000원
- 사무용품: 연 100,000원 (연 1회 지급)

지원금은 매월 급여와 함께 지급됩니다.

## 5. 보안 규정

### 5.1 정보 보안

재택근무 시 회사 정보 보안에 각별히 유의해야 합니다:

- **VPN 필수**: 모든 회사 시스템 접속 시 VPN 연결
- **화면 잠금**: 자리를 비울 때 반드시 화면 잠금
- **문서 출력 금지**: 기밀 문서 가정 내 출력 금지
- **외부인 주의**: 가족 등 외부인에게 화면 노출 주의

### 5.2 단말 보안

- 회사 지급 노트북만 사용
- 개인 USB 사용 금지
- 보안 프로그램 항상 실행
- OS/보안 업데이트 즉시 적용

### 5.3 보안 사고 발생 시

보안 사고(분실, 해킹 의심 등) 발생 시 즉시 보고:
- 보안팀: security@techflow.kr
- 긴급 연락처: 010-1234-5678 (24시간)

## 6. 재택근무와 다른 제도 병행

### 6.1 재택근무 + 휴가

재택근무일에 휴가(반차 포함)를 사용할 수 있습니다.

- 오전 반차: 13:00부터 재택 출근 체크
- 오후 반차: 13:00 재택 퇴근 체크

### 6.2 재택근무 + 병가

재택근무일에 병가가 필요한 경우:
1. 팀장에게 연락 (09:00 이전)
2. HR 시스템에서 재택 취소 + 병가 신청
3. 3일 이상 시 진단서 제출

**중요**: 컨디션이 좋지 않은 상태에서 무리하게 재택근무하지 말고, 병가를 사용하세요.

### 6.3 재택근무 + 탄력근무

탄력근무제와 재택근무를 함께 사용할 수 있습니다.

탄력근무 유형:
- 시차출퇴근: 08:00~17:00 또는 10:00~19:00
- 집중근무: 주 4일 10시간 근무

신청: HR Portal에서 재택근무와 탄력근무 모두 신청

## 7. 주의사항

### 7.1 재택근무 남용 금지

다음의 경우 재택근무 자격이 박탈될 수 있습니다:

- 연락 불가 상태가 반복되는 경우
- 업무 성과가 현저히 저하된 경우
- 보안 규정을 위반한 경우
- 허위로 재택근무를 신청한 경우

### 7.2 외부 장소 근무

재택근무는 원칙적으로 "자택"에서 수행합니다.

- 카페, 도서관 등에서의 근무는 보안상 금지
- 불가피한 경우 사전 승인 필요
- 해외에서의 근무는 별도 절차 필요

### 7.3 건강 관리

재택근무 시 건강 관리에 유의하세요:

- 1시간마다 스트레칭
- 충분한 수분 섭취
- 눈 건강 (20-20-20 규칙: 20분마다 20피트 거리를 20초간 바라보기)
- 정해진 시간에 출퇴근하여 워라밸 유지

## 8. FAQ

**Q: 재택근무일에 외부 미팅이 잡히면 어떻게 하나요?**
A: 외부 미팅 참석 후 재택 또는 사무실 복귀 중 선택하세요. 이동시간은 근무시간으로 인정됩니다.

**Q: 인터넷 장애로 업무가 불가능하면?**
A: 즉시 팀장에게 연락하고, 가능하면 모바일 핫스팟으로 최소한의 연락을 유지하세요. 장애가 길어지면 출근합니다.

**Q: 재택근무 중 야근이 필요하면?**
A: 사전에 팀장 승인을 받고, 야근 시간을 기록합니다. 야근 식대는 영수증 제출로 정산됩니다.
