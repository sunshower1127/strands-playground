# FlowSync 트러블슈팅 가이드

> 버전: 2.5.0
> 최종 수정: 2024년 10월 20일
> 담당: 기술지원팀 (support@flowsync.io)

## 1. 자주 발생하는 문제

### 1.1 로그인 문제

#### 증상: 로그인 버튼 클릭 시 무한 로딩

**원인**:
- 브라우저 캐시 문제
- 네트워크 연결 불안정
- SSO 서버 응답 지연

**해결 방법**:
1. 브라우저 캐시 삭제: 설정 > 개인정보 > 인터넷 사용 기록 삭제
2. 시크릿/프라이빗 모드에서 재시도
3. 다른 브라우저에서 시도 (Chrome, Safari, Firefox 권장)
4. 회사 VPN 연결 상태 확인

#### 증상: "잘못된 자격 증명" 오류

**원인**:
- 비밀번호 오타
- 계정 비활성화
- SSO 연동 계정으로 일반 로그인 시도

**해결 방법**:
1. Caps Lock 확인
2. [비밀번호 재설정] 링크 클릭
3. SSO 계정인 경우 "회사 계정으로 로그인" 사용
4. 계정 상태 문의: support@flowsync.io

#### 증상: 2단계 인증 코드 오류

**해결 방법**:
1. 휴대폰 시간 동기화 확인 (자동 설정)
2. 인증 앱 재등록
3. 백업 코드 사용 (초기 설정 시 저장)
4. IT팀에 2FA 초기화 요청

### 1.2 동기화 문제

#### 증상: 파일 동기화가 멈춤

**원인**:
- 네트워크 연결 끊김
- 저장 공간 부족
- 파일 사용 중 (다른 프로그램에서 열림)
- 동기화 충돌

**해결 방법**:

**Step 1: 상태 확인**
- 시스템 트레이 > FlowSync 아이콘 우클릭 > [동기화 상태]
- 오류 메시지 확인

**Step 2: 기본 조치**
1. FlowSync 앱 재시작
2. 네트워크 연결 확인
3. 저장 공간 확인 (최소 1GB 여유 필요)

**Step 3: 충돌 해결**
- 충돌 파일 우클릭 > [충돌 해결]
- 로컬 유지 / 서버 유지 / 둘 다 유지 선택

**Step 4: 강제 동기화**
- 시스템 트레이 > FlowSync > [지금 동기화]
- 또는 문제 폴더 우클릭 > [동기화 재시도]

#### 증상: 파일 동기화 오류 (특정 파일만)

**원인**:
- 파일명에 특수문자 포함
- 파일 크기 제한 초과 (10GB)
- 파일 경로 길이 초과 (260자)
- 파일 권한 문제

**해결 방법**:

1. **파일명 확인**: 다음 특수문자 제거
   - 사용 불가: `\ / : * ? " < > |`
   - 권장: 영문, 숫자, 한글, 공백, `-`, `_`

2. **파일 크기 확인**: 10GB 초과 시 분할 압축

3. **경로 길이 확인**:
   ```
   전체 경로 = C:\Users\...\FlowSync\폴더\파일명.확장자
   260자 이내로 유지
   ```

4. **권한 확인**: 파일 속성 > 보안 탭에서 읽기/쓰기 권한 확인

#### 증상: 동기화 속도가 매우 느림

**해결 방법**:
1. 대역폭 제한 해제: 설정 > 네트워크 > 업로드/다운로드 제한 해제
2. 선택적 동기화 활성화: 불필요한 폴더 동기화 제외
3. 바이러스 백신 제외 설정: FlowSync 폴더를 검사 제외에 추가
4. 네트워크 상태 점검: 다른 파일 다운로드로 속도 테스트

### 1.3 문서 편집 문제

#### 증상: 공동 편집 시 다른 사람 커서가 안 보임

**원인**:
- 실시간 연결 끊김
- 브라우저 WebSocket 차단

**해결 방법**:
1. 페이지 새로고침 (F5)
2. 브라우저 확장 프로그램 일시 비활성화 (광고 차단 등)
3. 회사 방화벽에서 WebSocket 허용 확인 (IT팀 문의)

#### 증상: 문서 저장 실패

**원인**:
- 네트워크 일시 끊김
- 동시 편집 충돌
- 저장 공간 초과

**해결 방법**:
1. 변경 내용 복사 (Ctrl+A, Ctrl+C)
2. 새로고침 후 다시 붙여넣기
3. 버전 기록에서 마지막 저장 버전 확인
4. 저장 공간 확인 및 정리

#### 증상: 마크다운 렌더링 안 됨

**해결 방법**:
1. 편집 모드 확인: [보기] > [마크다운 미리보기]
2. 문법 확인: 공백/줄바꿈 정확히 입력
3. 특수 문자 이스케이프: `\*`, `\_` 등

## 2. API 오류

### 2.1 인증 오류 (401)

#### 증상: "Invalid or expired API key"

**해결 방법**:
1. API 키 형식 확인: `fs_live_` 또는 `fs_test_` 로 시작
2. 키 만료 여부 확인: 설정 > 통합 > API에서 확인
3. 환경 변수 설정 확인:
   ```bash
   echo $FLOWSYNC_API_KEY
   ```
4. 새 API 키 발급

### 2.2 권한 오류 (403)

#### 증상: "Access denied" 또는 "Forbidden"

**원인**:
- API 키 권한 부족
- 리소스 접근 권한 없음
- IP 화이트리스트 제한

**해결 방법**:
1. API 키 권한 범위 확인
2. 리소스 소유자에게 공유 요청
3. 관리자에게 IP 화이트리스트 추가 요청

### 2.3 Rate Limit 오류 (429)

#### 증상: "Rate limit exceeded"

**해결 방법**:
1. 요청 간격 조절 (지수 백오프 적용)
   ```python
   import time

   def retry_with_backoff(func, max_retries=5):
       for i in range(max_retries):
           try:
               return func()
           except RateLimitError:
               wait = 2 ** i  # 1, 2, 4, 8, 16초
               time.sleep(wait)
       raise Exception("Max retries exceeded")
   ```

2. 배치 API 활용 (여러 작업 한 번에)
3. 캐싱 도입 (동일 요청 반복 방지)
4. 플랜 업그레이드 검토

### 2.4 서버 오류 (500)

#### 증상: "Internal server error"

**해결 방법**:
1. 몇 분 후 재시도
2. 요청 파라미터 검증
3. 상태 페이지 확인: status.flowsync.io
4. 지속 시 지원팀 문의 (요청 ID 포함)

## 3. 데스크톱 앱 문제

### 3.1 앱 실행 안 됨

#### Windows

**증상**: 앱 아이콘 클릭해도 무반응

**해결 방법**:
1. 작업 관리자에서 FlowSync 프로세스 종료
2. `%APPDATA%\FlowSync\` 폴더 삭제 (설정 초기화)
3. 앱 재설치

**증상**: "이 앱은 PC에서 실행할 수 없습니다"

**해결 방법**:
1. 최신 버전 다운로드 (flowsync.io/download)
2. 관리자 권한으로 설치
3. Windows 버전 확인 (10 이상 필요)

#### macOS

**증상**: "앱이 손상되었습니다"

**해결 방법**:
```bash
xattr -cr /Applications/FlowSync.app
```

**증상**: 앱이 응답하지 않음

**해결 방법**:
1. Option + Cmd + Esc로 강제 종료
2. `~/Library/Application Support/FlowSync/` 삭제
3. 재설치

### 3.2 트레이 아이콘 안 보임

**Windows**:
1. 작업 표시줄 > 숨겨진 아이콘 확인 (^)
2. 설정 > 개인 설정 > 작업 표시줄 > 시스템 트레이

**macOS**:
1. 메뉴 바 확인 (우측 상단)
2. 시스템 설정 > 제어 센터 > 메뉴 막대

### 3.3 업데이트 실패

**해결 방법**:
1. 수동 업데이트: 앱 > 도움말 > 업데이트 확인
2. 완전 재설치:
   - 기존 앱 삭제
   - 설정 폴더 삭제
   - 최신 버전 다운로드 및 설치
3. 방화벽/프록시 설정 확인

## 4. 모바일 앱 문제

### 4.1 알림 안 옴

**iOS**:
1. 설정 > FlowSync > 알림 > 허용
2. 집중 모드 확인
3. 앱 재설치

**Android**:
1. 설정 > 앱 > FlowSync > 알림 > 활성화
2. 배터리 최적화 제외
3. 백그라운드 데이터 허용

### 4.2 오프라인 모드 작동 안 함

**해결 방법**:
1. 오프라인 동기화할 파일 사전 선택 확인
2. 저장 공간 확인 (최소 500MB)
3. 앱 내 설정 > 오프라인 > 동기화 실행

### 4.3 카메라/갤러리 접근 안 됨

**해결 방법**:
1. 앱 권한 설정에서 카메라/사진 권한 허용
2. iOS: 설정 > 개인정보 보호 > 사진 > FlowSync
3. Android: 설정 > 앱 > FlowSync > 권한

## 5. 성능 최적화

### 5.1 웹 앱이 느림

**해결 방법**:
1. 브라우저 캐시 정리
2. 확장 프로그램 비활성화
3. 하드웨어 가속 활성화 (Chrome: 설정 > 고급 > 시스템)
4. 다른 탭 닫기 (메모리 확보)

### 5.2 대용량 파일 업로드 느림

**해결 방법**:
1. 청크 업로드 API 사용 (100MB 이상)
2. 압축 후 업로드
3. 네트워크 대역폭 확인
4. 피크 시간대 피해서 업로드

### 5.3 검색 결과 느림

**해결 방법**:
1. 검색 범위 좁히기 (특정 폴더/프로젝트)
2. 필터 활용 (파일 타입, 날짜 범위)
3. 인덱싱 완료 대기 (새 파일 업로드 직후)

## 6. 보안 문제

### 6.1 의심스러운 로그인 알림

**즉시 조치**:
1. 내 활동이 아니면 [아니요, 내가 아닙니다] 클릭
2. 즉시 비밀번호 변경
3. 모든 세션 로그아웃: 설정 > 보안 > 세션 관리
4. 2단계 인증 활성화

### 6.2 파일 무단 공유 의심

**확인 방법**:
1. 파일 우클릭 > 공유 설정 확인
2. 공유 링크 비활성화
3. 감사 로그 확인 (관리자 권한)

### 6.3 API 키 노출

**즉시 조치**:
1. 설정 > API > 해당 키 [폐기]
2. 새 API 키 발급
3. 코드/설정에서 새 키로 교체
4. 보안팀에 보고

## 7. 데이터 복구

### 7.1 삭제된 파일 복구

**휴지통 복구** (30일 이내):
1. FlowDrive > 휴지통
2. 파일 선택 > [복원]

**버전 복구**:
1. 파일 우클릭 > 버전 기록
2. 원하는 버전 선택 > [복원]

**완전 삭제된 파일**:
- 휴지통 비운 후 30일 이내: 지원팀 문의
- 30일 이후: 복구 불가

### 7.2 덮어쓴 파일 복구

**버전 기록 활용**:
1. 파일 우클릭 > 버전 기록
2. 이전 버전 확인 및 복원

### 7.3 계정 복구

**비활성화된 계정**:
- IT팀 또는 관리자에게 복구 요청
- 90일 이후 완전 삭제 (복구 불가)

## 8. 지원 요청

### 8.1 문의 전 준비사항

지원 요청 시 다음 정보를 준비해주세요:

1. **문제 설명**: 언제, 어떤 상황에서 발생하는지
2. **재현 단계**: 문제를 재현하는 구체적 단계
3. **환경 정보**:
   - OS 버전
   - 브라우저/앱 버전
   - FlowSync 버전
4. **스크린샷/화면 녹화**
5. **오류 메시지**: 정확한 오류 텍스트
6. **시도한 해결 방법**

### 8.2 지원 채널

| 채널 | 용도 | 응답 시간 |
|-----|-----|---------|
| 도움말 센터 | 가이드, FAQ | 즉시 |
| 이메일 | 일반 문의 | 24시간 내 |
| 긴급 전화 | 서비스 중단 | 1시간 내 |
| 채팅 | 빠른 질문 | 실시간 (업무시간) |

**연락처**:
- 이메일: support@flowsync.io
- 긴급: 02-1234-5678 (Enterprise 전용)
- 채팅: app.flowsync.io > 우측 하단 채팅 아이콘

### 8.3 로그 수집

기술 지원 요청 시 로그가 필요할 수 있습니다.

**데스크톱 앱**:
- Windows: `%APPDATA%\FlowSync\logs\`
- macOS: `~/Library/Logs/FlowSync/`

**수집 방법**:
1. 시스템 트레이 > FlowSync > [도움말]
2. [진단 보고서 생성] 클릭
3. 생성된 zip 파일 첨부
