## 1. FlowSync 소개

### 1.1 FlowSync란?

FlowSync는 TechFlow Inc.에서 개발한 올인원 협업 플랫폼입니다. 팀 커뮤니케이션, 프로젝트 관리, 파일 공유를 하나의 도구로 통합하여 업무 효율을 극대화합니다.

### 1.2 주요 기능 3가지

FlowSync의 핵심 기능은 다음 3가지입니다:

1. **실시간 협업 문서 (FlowDocs)**
   - Google Docs와 유사한 실시간 공동 편집
   - 버전 관리 및 변경 이력 추적
   - 마크다운 지원

2. **스마트 태스크 관리 (FlowTask)**
   - 칸반 보드 및 간트 차트 뷰
   - 자동화된 워크플로우 설정
   - 의존성 관리 및 마일스톤 추적

3. **통합 파일 동기화 (FlowDrive)**
   - 크로스 플랫폼 파일 동기화 (Windows, Mac, Linux, Mobile)
   - 선택적 동기화로 저장공간 최적화
   - 오프라인 모드 지원

### 1.3 지원 플랫폼

| 플랫폼 | 지원 버전 | 다운로드 |
|-------|---------|---------|
| Windows | 10, 11 | flowsync.io/download/win |
| macOS | 12 이상 | flowsync.io/download/mac |
| Linux | Ubuntu 20.04+ | flowsync.io/download/linux |
| iOS | 15 이상 | App Store |

## 2. FlowDocs - 실시간 협업 문서

### 2.1 문서 생성

새 문서를 생성하는 방법:

1. 좌측 사이드바에서 [+ 새로 만들기] 클릭
2. [FlowDoc] 선택
3. 템플릿 선택 또는 빈 문서로 시작
4. 문서 제목 입력

**단축키**: `Ctrl/Cmd + N`

### 2.2 공동 편집

FlowDocs는 최대 50명이 동시에 편집할 수 있습니다.

**실시간 편집 기능**:
- 다른 사용자의 커서 위치 실시간 표시 (색상으로 구분)
- 편집 중인 블록 자동 잠금 (충돌 방지)
- 변경사항 즉시 반영 (지연 시간 < 100ms)

**댓글 및 제안**:
- 텍스트 선택 후 우클릭 > [댓글 추가]
- 제안 모드: 변경사항을 제안으로 표시하고 승인/거절 가능
- @멘션으로 팀원 호출

### 2.3 버전 관리

모든 문서는 자동으로 버전이 관리됩니다.

- **자동 저장**: 변경 시 즉시 저장
- **버전 이력**: 지난 30일간 모든 버전 보관
- **버전 비교**: 두 버전 간 차이점 하이라이트
- **복원**: 이전 버전으로 원클릭 복원

버전 이력 확인: [파일] > [버전 기록] 또는 `Ctrl/Cmd + Shift + H`

### 2.4 마크다운 지원

FlowDocs는 마크다운 문법을 지원합니다.

```markdown
# 제목 1
## 제목 2
### 제목 3

**굵게** *기울임* ~~취소선~~

- 목록 1
- 목록 2
  - 하위 목록

1. 번호 목록
2. 번호 목록

> 인용문

`인라인 코드`

​```python
코드 블록
​```

[링크](https://example.com)
![이미지](image_url)
```

### 2.5 내보내기

문서를 다양한 형식으로 내보낼 수 있습니다:

- PDF (인쇄용)
- DOCX (Word 호환)
- HTML (웹 게시용)
- Markdown (개발자용)

[파일] > [내보내기] > 형식 선택

## 3. FlowTask - 스마트 태스크 관리

### 3.1 프로젝트 생성

새 프로젝트 생성:

1. 좌측 [프로젝트] 메뉴 > [+ 프로젝트 만들기]
2. 프로젝트명 및 설명 입력
3. 팀원 초대
4. 뷰 타입 선택 (칸반/리스트/간트/캘린더)

### 3.2 태스크 관리

**태스크 생성**:
- [+ 태스크 추가] 버튼 또는 `Ctrl/Cmd + Enter`
- 제목, 담당자, 마감일, 우선순위 설정

**태스크 속성**:
| 속성 | 설명 |
|-----|-----|
| 담당자 | 태스크 책임자 (다중 선택 가능) |
| 마감일 | 완료 기한 |
| 우선순위 | 긴급/높음/보통/낮음 |
| 상태 | 할 일/진행 중/검토 중/완료 |
| 태그 | 분류용 라벨 (다중 선택) |
| 예상 시간 | 작업 소요 시간 추정치 |
| 하위 태스크 | 세부 작업 분할 |

### 3.3 칸반 보드

드래그 앤 드롭으로 태스크 상태를 변경합니다.

기본 칼럼:
- **할 일 (To Do)**: 시작 전 태스크
- **진행 중 (In Progress)**: 작업 중인 태스크
- **검토 중 (In Review)**: 검토 대기 중
- **완료 (Done)**: 완료된 태스크

칼럼 커스터마이징: [설정] > [보드 칼럼 관리]

### 3.4 간트 차트

프로젝트 타임라인을 시각적으로 관리합니다.

**간트 차트 기능**:
- 마감일 기반 타임라인 자동 생성
- 태스크 간 의존성 설정 (화살표로 연결)
- 마일스톤 표시
- 크리티컬 패스 하이라이트
- 리소스 할당 뷰

의존성 설정: 태스크 드래그하여 다른 태스크에 연결

### 3.5 자동화 규칙

반복적인 작업을 자동화합니다.

**예시 자동화 규칙**:

```
IF 태스크 상태가 "완료"로 변경되면
THEN 담당자에게 알림 전송
AND 다음 태스크 담당자에게 할당 알림
```

```
IF 마감일이 1일 남았으면
THEN Slack 채널에 알림 전송
```

```
IF 새 태스크가 "버그" 태그로 생성되면
THEN 우선순위를 "높음"으로 설정
AND QA 팀에게 할당
```

자동화 설정: [프로젝트 설정] > [자동화]

### 3.6 리포팅

프로젝트 현황을 한눈에 파악합니다.

**제공 리포트**:
- 번다운 차트 (Burndown Chart)
- 속도 차트 (Velocity Chart)
- 태스크 완료율
- 팀원별 작업량
- 지연 태스크 리스트

리포트 확인: [프로젝트] > [리포트] 탭

## 4. FlowDrive - 통합 파일 동기화

### 4.1 파일 업로드

파일을 FlowDrive에 업로드하는 방법:

**웹/데스크톱**:
- 드래그 앤 드롭
- [+ 업로드] 버튼 클릭
- 단축키: `Ctrl/Cmd + U`

**모바일**:
- [+] 버튼 > [파일 업로드]
- 갤러리에서 사진/동영상 선택

**업로드 제한**:
- 단일 파일: 최대 10GB
- 총 저장공간: 플랜에 따라 상이 (Free: 5GB, Pro: 100GB, Enterprise: 무제한)

### 4.2 폴더 구조

효율적인 파일 관리를 위한 폴더 구조:

```
FlowDrive/
├── 내 파일/           # 개인 파일
├── 공유 문서/          # 팀 공유 파일
│   ├── 프로젝트 A/
│   ├── 프로젝트 B/
│   └── 참고 자료/
├── 휴지통/            # 삭제된 파일 (30일 보관)
└── 즐겨찾기/          # 자주 사용하는 파일
```

### 4.3 파일 동기화

데스크톱 앱 설치 시 로컬 폴더와 자동 동기화됩니다.

**동기화 설정**:
1. FlowSync 데스크톱 앱 설치
2. 시스템 트레이 아이콘 클릭
3. [동기화 설정] 선택
4. 동기화할 폴더 선택 (선택적 동기화)

**동기화 상태**:
- ✅ 초록색 체크: 동기화 완료
- 🔄 파란색 화살표: 동기화 중
- ⚠️ 노란색 경고: 충돌 발생
- ❌ 빨간색 X: 오류

### 4.4 오프라인 모드

인터넷 연결 없이도 파일 작업이 가능합니다.

**오프라인 사용 설정**:
1. 오프라인으로 사용할 파일/폴더 우클릭
2. [오프라인 사용 가능] 선택
3. 파일이 로컬에 다운로드됨

오프라인에서 수정한 파일은 인터넷 연결 시 자동 동기화됩니다.

### 4.5 파일 공유

파일을 외부 사용자와 공유합니다.

**공유 방법**:
1. 파일/폴더 우클릭 > [공유]
2. 공유 대상 선택:
   - 팀원: 이메일 또는 이름으로 검색
   - 외부 사용자: 이메일 입력
   - 링크 공유: 공개 링크 생성

**권한 설정**:
| 권한 | 설명 |
|-----|-----|
| 보기만 | 다운로드 불가, 조회만 가능 |
| 댓글 가능 | 조회 + 댓글 |
| 편집 가능 | 조회 + 편집 |
| 전체 권한 | 편집 + 공유 + 삭제 |

### 4.6 버전 관리

파일의 이전 버전을 확인하고 복원합니다.

- 모든 파일 변경 시 자동 버전 저장
- 최대 100개 버전 또는 30일간 보관
- 버전 이력: 파일 우클릭 > [버전 기록]

## 5. 통합 및 연동

### 5.1 기본 연동

FlowSync는 다음 서비스와 기본 연동됩니다:

- **Slack**: 알림 전송, 파일 공유
- **Google Workspace**: 캘린더 동기화, 드라이브 연결
- **Microsoft 365**: Outlook 캘린더, OneDrive 연결
- **GitHub**: 커밋과 태스크 연결
- **Jira**: 이슈 동기화
- **Figma**: 디자인 파일 미리보기

### 5.2 Webhook

커스텀 통합을 위한 Webhook을 제공합니다.

설정: [설정] > [통합] > [Webhook]

지원 이벤트:
- 태스크 생성/수정/완료
- 파일 업로드/수정/삭제
- 댓글 추가
- 멤버 초대/퇴장

### 5.3 API

FlowSync API로 커스텀 자동화를 구현합니다.

API 문서: api.flowsync.io/docs

## 6. 보안

### 6.1 데이터 암호화

- 전송 중 암호화: TLS 1.3
- 저장 시 암호화: AES-256

### 6.2 접근 제어

- SSO (Single Sign-On) 지원: Google, Okta, Azure AD
- 2단계 인증 (2FA) 필수 설정 가능
- IP 화이트리스트
- 세션 타임아웃 설정

### 6.3 감사 로그

모든 활동이 기록되며, 관리자가 조회할 수 있습니다.

[관리자] > [보안] > [감사 로그]
