# FlowSync 변경 이력 (Changelog)

> 최종 업데이트: 2024년 10월 25일

## v2.5.1 (2024.10.25) - Latest

### 🐛 버그 수정

- **파일 동기화**: 특정 조건에서 동기화가 멈추는 문제 수정
  - 영향: Windows 클라이언트에서 100MB 이상 파일 업로드 시
  - 원인: 청크 업로드 타임아웃 설정 오류
  - 해결: 타임아웃 값 조정 (30초 → 60초)

- **FlowDocs**: 공동 편집 시 커서 위치 동기화 지연 수정
  - 영향: 5명 이상 동시 편집 시
  - 개선: 지연 시간 500ms → 100ms

- **모바일 앱**: iOS에서 푸시 알림이 오지 않는 문제 수정
  - 영향: iOS 17.1 이상 사용자
  - 원인: APNs 인증서 갱신 누락

### 🔧 개선

- 대시보드 로딩 속도 15% 개선
- 검색 결과 정확도 향상
- 메모리 사용량 최적화

## v2.3.0 (2024.08.15)

### ✨ 새로운 기능

- **FlowSync AI Assistant** 🤖
  - 자연어로 태스크 생성: "내일까지 보고서 작성해줘"
  - 문서 검색: "지난주 회의록 찾아줘"
  - 일정 요약: "이번 주 미팅 정리해줘"
  - 질문 답변: 제품 사용법, 회사 정책 등

- **AI 활용 예시**
  ```
  사용자: "김개발님이 담당한 완료 안 된 태스크 보여줘"
  AI: [태스크 목록 표시]

  사용자: "API 문서에서 인증 방법 찾아줘"
  AI: [관련 문서 섹션 표시]
  ```

### 🔧 개선

- AI 응답 속도 최적화 (평균 2.5초 → 1.8초)
- 대화 문맥 유지 개선

### 📝 알려진 이슈

- 복잡한 질문 이해도 개선 필요 (향후 업데이트 예정)

## v2.0.0 (2024.04.30) ⭐ Major Release

### ✨ 새로운 기능

- **FlowSync API v2**
  - RESTful API 전면 개편
  - GraphQL 지원 추가
  - 향상된 Rate Limiting
  - Webhook 이벤트 확장

- **API 주요 변경**
  - 엔드포인트 구조 통일: `/v2/resources/{id}`
  - 응답 형식 표준화
  - 에러 코드 체계화

### ⚠️ Breaking Changes

- API v1 Deprecated (2024.10.31 종료 예정)
- 기존 Webhook 형식 변경 (마이그레이션 가이드 제공)

### 📖 마이그레이션 가이드

API v1 → v2 마이그레이션: docs.flowsync.io/migration/v2
